# 🌦️ BreezeTrack - Weather Forecasting Application

## Overview

BreezeTrack is a responsive weather forecasting web application that provides real-time weather information for any location. The application fetches live weather data using the OpenWeather API and displays essential weather details such as temperature, humidity, wind speed, and weather conditions in a user-friendly interface.

The project was developed to provide users with quick and accurate weather updates while offering a seamless experience across desktop and mobile devices.

---

## Features

* Real-time weather updates
* Location-based weather information
* Search weather by city name
* Temperature, humidity, and wind speed display
* Dynamic weather condition updates
* Error handling for invalid locations
* Responsive design for mobile and desktop devices
* Clean and intuitive user interface

## How It Works

1. User enters a city name or allows location access.
2. The application sends a request to the OpenWeather API.
3. Weather data is fetched in JSON format.
4. JavaScript processes the response.
5. Weather details are displayed dynamically on the user interface.
